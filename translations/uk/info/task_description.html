Вам задано список координат вершин трикутників (список кортежів з 3 кортежів по 2 цілих числа), на які розбито многокутник. Ці трикутники мають хоча б одну спільну сторону з іншим трикутником (який також має дві спільні вершини). Ви повинні повернути кількість сторін вихідного многокутника як ціле число.

Приклади: 

assert number_of_sides([((1, 1), (1, 3), (3, 3)), ((1, 1), (3, 1), (3, 3))]) == 4
assert number_of_sides([((1, 1), (2, 1), (2, 3)), ((2, 1), (2, 3), (3, 1))]) == 3
assert number_of_sides([((1, 1), (2, 3), (3, 1)), ((2, 3), (3, 1), (4, 3)),
                        ((2, 3), (3, 5), (4, 3)), ((3, 1), (4, 3), (5, 1))]) == 3

ПРИМІТКА:

  • Даний многокутник не має отворів.

Вхідні дані: Список координат трикутників (список кортежів з 3 кортежів по 2 цілих числа) ({% if is_js %}number{% else %}int{% endif %}).

Вихідні дані: Кількість сторін (ціле число) ({% if is_js %}number{% else %}int{% endif %}).

Передумова:

  • x >= 0, y >= 0
