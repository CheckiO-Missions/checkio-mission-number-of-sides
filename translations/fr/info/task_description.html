On vous donne une liste de coordonnées des sommets des triangles (une liste de tuples de 3 tuples de 2 entiers) en lesquels un polygone est divisé. Ces triangles partagent au moins un côté avec l'autre triangle (qui partage également deux sommets). Vous devez renvoyer le nombre de côtés du polygone original sous la forme d'un entier.

Exemples: 

assert number_of_sides([((1, 1), (1, 3), (3, 3)), ((1, 1), (3, 1), (3, 3))]) == 4
assert number_of_sides([((1, 1), (2, 1), (2, 3)), ((2, 1), (2, 3), (3, 1))]) == 3
assert number_of_sides([((1, 1), (2, 3), (3, 1)), ((2, 3), (3, 1), (4, 3)),
                        ((2, 3), (3, 5), (4, 3)), ((3, 1), (4, 3), (5, 1))]) == 3

NOTE:

  • Ce polygone n'a pas de trous.

Entrée: La liste des coordonnées des triangles (Une liste de tuples de 3 tuples de 2 entiers) ({% if is_js %}number{% else %}int{% endif %}).

Sortie: Le nombre de côtés (Un entier) ({% if is_js %}number{% else %}int{% endif %}).

Condition préalable:

  • x >= 0, y >= 0
