给您一个三角形顶点坐标列表（一个由 3 个 2 整数组成的元组列表），一个多边形被分割成多个三角形。 这些三角形至少有一条边与另一个三角形共享（另一个三角形也共享两个顶点）。 您必须以整数形式返回原始多边形的边数。

示例 

assert number_of_sides([((1, 1), (1, 3), (3, 3)), ((1, 1), (3, 1), (3, 3))]) == 4
assert number_of_sides([((1, 1), (2, 1), (2, 3)), ((2, 1), (2, 3), (3, 1))]) == 3
assert number_of_sides([((1, 1), (2, 3), (3, 1)), ((2, 3), (3, 1), (4, 3)),
                        ((2, 3), (3, 5), (4, 3)), ((3, 1), (4, 3), (5, 1))]) == 3

注意：

  • 该多边形没有孔。

输入三角形坐标列表（由 3 个元组组成的列表，其中 3 个元组为 2 个整数） ({% if is_js %}number{% else %}int{% endif %}).

输出：边数（一个整数） ({% if is_js %}number{% else %}int{% endif %}).

先决条件

  • x >= 0, y >= 0
