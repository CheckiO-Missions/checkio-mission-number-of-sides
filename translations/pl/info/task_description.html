Otrzymują Państwo listę współrzędnych wierzchołków trójkątów (lista krotek po 3 krotki po 2 liczby całkowite), na które podzielony jest wielokąt. Trójkąty te dzielą co najmniej jeden bok z drugim trójkątem (który również dzieli dwa wierzchołki). Proszę zwrócić liczbę boków oryginalnego wielokąta jako liczbę całkowitą.

Przykłady: 

assert number_of_sides([((1, 1), (1, 3), (3, 3)), ((1, 1), (3, 1), (3, 3))]) == 4
assert number_of_sides([((1, 1), (2, 1), (2, 3)), ((2, 1), (2, 3), (3, 1))]) == 3
assert number_of_sides([((1, 1), (2, 3), (3, 1)), ((2, 3), (3, 1), (4, 3)),
                        ((2, 3), (3, 5), (4, 3)), ((3, 1), (4, 3), (5, 1))]) == 3

UWAGA:

  • Ten wielokąt nie ma otworów.
