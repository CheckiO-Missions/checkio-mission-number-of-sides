Sie erhalten eine Liste mit den Koordinaten der Eckpunkte der Dreiecke (eine Liste von Tupeln aus 3 Tupeln mit 2 ganzen Zahlen), in die ein Polygon unterteilt ist. Diese teilen sich mindestens eine Seite mit dem anderen Dreieck (das sich ebenfalls zwei Eckpunkte teilt). Sie müssen die Anzahl der Seiten des ursprünglichen Polygons als ganze Zahl zurückgeben.

Beispiele: 

assert number_of_sides([((1, 1), (1, 3), (3, 3)), ((1, 1), (3, 1), (3, 3))]) == 4
assert number_of_sides([((1, 1), (2, 1), (2, 3)), ((2, 1), (2, 3), (3, 1))]) == 3
assert number_of_sides([((1, 1), (2, 3), (3, 1)), ((2, 3), (3, 1), (4, 3)),
                        ((2, 3), (3, 5), (4, 3)), ((3, 1), (4, 3), (5, 1))]) == 3

HINWEIS:

  • Dieses Polygon hat keine Löcher.

Eingabe: Die Liste der Koordinaten der Dreiecke (Eine Liste von Tupeln aus 3 Tupeln mit 2 ganzen Zahlen) ({% if is_js %}number{% else %}int{% endif %}).

Ausgabe: Die Anzahl der Seiten (Eine ganze Zahl) ({% if is_js %}number{% else %}int{% endif %}).

Vorbedingung:

  • x >= 0, y >= 0
